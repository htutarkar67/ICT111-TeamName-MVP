# Lab 04 - User Persona

## 1. Persona Name

Alex Tan, 20, University Student

## 2. User Type

Student who enjoys watching movies but struggles to find movies that match his interests.

## 3. Background and Context

* Alex watches movies during weekends and free time.
* He often spends a long time browsing streaming platforms before deciding what to watch.
* He uses Netflix, Disney+, YouTube, and social media for movie recommendations.
* He prefers personalized suggestions rather than searching manually.

## 4. Goals

* Find movies that match his preferences quickly.
* Discover new movies and genres.
* Reduce the time spent searching for a movie.
* Enjoy a better movie-watching experience.

## 5. Pain Points from Lab 03 Evidence

| Pain Point                                    | Evidence ID / Respondent | Explanation                                                                  |
| --------------------------------------------- | ------------------------ | ---------------------------------------------------------------------------- |
| Too many movie choices available              | R01                      | User feels overwhelmed by the large number of movie options.                 |
| Takes too long to decide what to watch        | R03                      | User spends significant time browsing before selecting a movie.              |
| Existing recommendations are not personalized | R05                      | Recommendations often do not match the user's interests.                     |
| Difficult to discover new movies              | R07                      | User frequently watches the same movies due to lack of suitable suggestions. |

## 6. Current Alternatives / Workarounds

| Current Alternative             | Weakness / Limitation                                           | Evidence |
| ------------------------------- | --------------------------------------------------------------- | -------- |
| Netflix recommendations         | Recommendations can be repetitive and limited.                  | R01      |
| Asking friends for suggestions  | Recommendations may not match personal preferences.             | R04      |
| Google searches and movie blogs | Time-consuming and requires reading multiple reviews.           | R03      |
| Watching trending movies        | Popular movies are not always relevant to the user's interests. | R06      |

## 7. Design Implications

* The system should provide personalized movie recommendations.
* Users should be able to filter movies by genre, rating, release year, and language.
* The recommendation process should be simple and fast.
* The interface should be user-friendly and easy to navigate.
* The MVP should focus on helping users discover suitable movies quickly based on their preferences.

Lab 04 - User Persona
1. Persona Name

Alex Tan, 20, University Student

2. User Type

Movie watcher who struggles to find movies matching personal interests

3. Background and Context
Alex watches movies and TV shows during free time, especially on weekends.
He often spends more time searching for a movie than actually watching one.
He uses streaming platforms such as Netflix, Disney+, and YouTube.
He usually relies on social media recommendations or friends' suggestions.
4. Goals
Find movies that match his interests quickly.
Discover new movies without spending a long time searching.
Receive personalized recommendations based on favorite genres.
Save time when deciding what to watch.
5. Pain Points from Lab 03 Evidence
Pain Point	Evidence ID / Respondent	Explanation
Too many movie choices	R01	User feels overwhelmed by the large number of available movies.
Takes a long time to decide	R03	User spends 20–30 minutes browsing before choosing a movie.
Recommendations are not personalized	R05	User receives suggestions that do not match preferences.
Difficult to discover new movies	R07	User repeatedly watches familiar movies because finding new ones is difficult.
6. Current Alternatives / Workarounds
Current Alternative	Weakness / Limitation	Evidence
Netflix recommendations	Recommendations are often repetitive and limited.	R01
Asking friends for suggestions	Recommendations depend on friends' tastes.	R04
Searching on Google	Time-consuming and requires reading many reviews.	R03
Watching trending movies	Popular movies may not match personal preferences.	R06
7. Design Implications
The system should provide personalized movie recommendations.
The system should allow filtering by genre, rating, release year, and language.
The recommendation process should be simple and fast.
The interface should be easy to use for first-time users.
Avoid showing too many options at once.
The most important MVP feature is personalized movie recommendations based on user preferences.
